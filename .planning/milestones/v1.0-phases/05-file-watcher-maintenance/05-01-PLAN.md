**4. Add `indexFiles` to `src/core/indexer.ts`** — a batch helper that reindexes a specific list of file paths:

```typescript
export async function indexFiles(
  filePaths: string[],
  config: Config,
  db: Database.Database,
  table: lancedb.Table,
  embedder: EmbeddingProvider,
): Promise<IndexResult> {
  const result: IndexResult = {
    filesIndexed: 0, filesSkipped: 0, filesFailed: 0,
    chunksCreated: 0, failedPaths: [],
  };
  for (const filePath of filePaths) {
    let attempts = 0;
    while (attempts < 2) {
      attempts++;
      try {
        const fileResult = await indexFile(filePath, config, db, table, embedder);
        if (fileResult.status === 'indexed') result.filesIndexed++;
        else result.filesSkipped++;
        result.chunksCreated += fileResult.chunksCreated;
        break; // success — exit retry loop
      } catch (err) {
        if (attempts >= 2) {
          result.filesFailed++;
          result.failedPaths.push(filePath);
          logger.error(`Failed to index after ${attempts} attempts: ${filePath}`, err);
        }
      }
    }
  }
  return result;
}
```
Single retry with clear control flow: attempt once, on failure retry once, on second failure record as failed. No nested try/catch.

**`processBatch(...)` (internal, but exported for testing)**

Logic per CONTEXT.md locked decisions:
1. For each path in batch, check if file exists on disk (`fs.access`)
2. Existing files = adds/changes, missing files = unlinks
3. For unlinks: look up stored content hash in SQLite via `getFileHash(db, path)`
4. For adds: compute SHA-256 of file content
5. Match unlink hashes against add hashes — matches are renames
6. Renames: call `updateChunksSourcePath(table, oldPath, newPath)` + `updateSourcePath(db, oldPath, newPath)` — no embedding
7. Real deletes (unmatched unlinks): call `deleteChunksByPath(table, path)` + delete from SQLite files table
8. Real adds/changes (unmatched adds): call `indexFiles([...paths], config, db, table, embedder)`
9. Call `incrementUpdateCounter(db, reindexedCount)` if any files were reindexed
10. Wrap entire function in try/catch, log errors via logger.error, do NOT rethrow

**`startupCatchUp(options: WatcherOptions): Promise<{ reindexed: number; total: number }>`**

1. Call `scanVault(config)` to get all current vault files
2. For each file: compare `fs.stat(file).mtimeMs` against SQLite `indexed_at`. Skip if mtime <= indexed_at (fast pre-filter per RESEARCH open question 3).
3. For files where mtime is newer: compute SHA-256, compare against `getFileHash(db, path)`. If hash differs, add to staleFiles list.
4. If staleFiles non-empty: call `indexFiles(staleFiles, config, db, table, embedder)`
5. Log count to stderr via logger.info
6. Return `{ reindexed: staleFiles.length, total: vaultFiles.length }`
